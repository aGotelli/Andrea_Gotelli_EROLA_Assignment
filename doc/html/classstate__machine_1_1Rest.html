Detailed Description

This class defines the state of the state machine corresponding to the robot sleeping.

This class inheritates from smach and it consist of a state in the state machine. The state that is represented here is the Rest state. Where the pet like robot rests for recover energies. As part of the smach class, this class has the member function execute() providing the intended behavior. For more details about the content of this class, see the member function documentation.
